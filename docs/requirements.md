# 要件定義書 — ターミナル版インベーダゲーム (C++)

## 1. プロジェクト概要

| 項目 | 内容 |
|------|------|
| プロジェクト名 | InvaderGame |
| 実行環境 | Windows ターミナル（コマンドプロンプト / Windows Terminal） |
| 実装言語 | C++17 |
| ビルドシステム | CMake 3.20 以上 |
| レンダリング | Windows Console API (`windows.h`) |

---

## 2. ゲーム概要

スペースインベーダーを模したターミナル上で動くゲーム。
プレイヤーは自機を左右に動かしてミサイルを発射し、迫り来るインベーダー軍団を全滅させることを目指す。

---

## 3. 機能要件

### 3.1 画面・描画

| ID | 要件 |
|----|------|
| F-01 | ゲームフィールドはターミナル上に ASCII/ANSI 文字で描画する |
| F-02 | フレームレートは 30 FPS を目標とする |
| F-03 | 画面サイズは幅 80 × 高さ 25 文字（固定）とする |
| F-04 | ダブルバッファリングにより画面のちらつきを防ぐ |
| F-05 | スコア・残機数・レベルをステータスバーに常時表示する |

### 3.2 プレイヤー

| ID | 要件 |
|----|------|
| F-10 | プレイヤーは画面下部に配置し、左右キー（← →）で移動できる |
| F-11 | スペースキーでミサイルを発射する |
| F-12 | 同時に発射できるミサイルは最大 1 発とする |
| F-13 | 初期残機数は 3 とする |
| F-14 | インベーダーまたはインベーダーのミサイルに被弾すると残機を 1 減らす |
| F-15 | 残機が 0 になるとゲームオーバーとなる |

### 3.3 インベーダー

| ID | 要件 |
|----|------|
| F-20 | インベーダーは 5 列 × 3 行（計 15 体）の隊列を組んで配置する |
| F-21 | インベーダーは左右に移動し、端に到達すると一段下降して反対方向へ進む |
| F-22 | インベーダーがプレイヤーと同じ行まで降下した場合はゲームオーバーとなる |
| F-23 | インベーダーはランダムな間隔でミサイルを発射する |
| F-24 | インベーダーをすべて倒すとそのレベルをクリアとする |

### 3.4 スコア・レベル

| ID | 要件 |
|----|------|
| F-30 | インベーダーを 1 体倒すごとにスコアを加算する（行によって点数を変える） |
| F-31 | レベルクリアごとに次のレベルへ進む（レベルは 1 〜 5） |
| F-32 | レベルが上がるにつれてインベーダーの移動速度・発射頻度を増加させる |
| F-33 | レベル 5 クリアでゲームクリアとなる |
| F-34 | ハイスコアをファイル（`save/highscore.dat`）に保存・読み込みする |

### 3.5 UI・操作

| ID | 要件 |
|----|------|
| F-40 | タイトル画面（スタート / ハイスコア表示 / 終了 を選択） |
| F-41 | ポーズ機能（`P` キーでポーズ / 再開） |
| F-42 | ゲームオーバー画面（スコア表示・リトライ / タイトルへ 選択） |
| F-43 | ゲームクリア画面（スコア表示・リトライ / タイトルへ 選択） |

---

## 4. 非機能要件

| ID | 要件 |
|----|------|
| NF-01 | ビルド後の実行ファイル 1 つで動作する（外部ランタイム不要） |
| NF-02 | コード品質：クラス設計、ヘッダ分離、単一責任の原則を遵守する |
| NF-03 | メモリリークがないこと（動的確保は RAII / スマートポインタで管理） |
| NF-04 | Windows 10 / 11 のコマンドプロンプト・Windows Terminal で動作すること |
| NF-05 | CMake でビルドでき、`build/` ディレクトリに成果物を出力すること |

---

## 5. クラス設計（概要）

```
Game          ゲームループ・状態管理（タイトル/プレイ/ポーズ/終了）
Renderer      ダブルバッファリングによる描画管理
Player        プレイヤー座標・移動・ミサイル管理
Invader       インベーダー個体の状態・描画
InvaderSwarm  インベーダー集団の移動・射撃ロジック
Bullet        ミサイル（プレイヤー用・敵用）の移動と当たり判定
ScoreManager  スコア計算・ハイスコアの保存と読み込み
InputHandler  キー入力の非ブロッキング取得
```

## 7. 操作キー一覧

| キー | 動作 |
|------|------|
| ← / → | プレイヤー移動 |
| Space | ミサイル発射 |
| P | ポーズ / 再開 |
| Q / Esc | ゲーム終了（タイトルへ） |
| Enter | 選択確定 |
